# PixelMind Frontend Assignment

A responsive e-commerce product listing app.

## Features
- Product grid with search & filter
- Product details modal
- Shopping cart with item count
- Add Product form (mock)
- Responsive design

## Tech Stack
- Next.js
- Tailwind CSS
- React Query
- Zustand
- React Hook Form + Zod
